Overlays

When locating the route of a long-gone streetcar line, often the route traveled on private right of way which since has been obscured by subsequent construction. Also, street alignments may have been changed or eliminated due to highway construction, urban renewal or other activities. In such cases, having access to historical maps as an overlay will allow you to locate exactly where a route ran.

An overlay is a map that can be displayed with variable transparency on top of the main map (in Mapper either Google Maps or Open Street Map).


Overlay Management

Mappers overlays are managed with the Manage Overlays menu selection accessed from the City menu. This selection will display:

In this dialog, you can:

  1. alter the coordinates of the center
  2. alter properties on defined overlays
  3. select which overlay can be used for a city,
  4. add a new overlay by pressing the Add Overlay button

Adding Overlays

There are three ways to acquire overlays to use in Mapper.

  1. There are map servers such as the Georeferencer mentioned below which can be accessed via the Internet.
  2. If you have a web server on  your local network, you can have it serve map tiles.
  3. You can install a web server on your PC to serve map tiles.

Using Georeferencer to serve overlay map tiles.

One source of historical maps, in the US is the USGS. They have made many historical topographic maps available on-line, free of charge, at    USGS Map Store.

While historical topographic maps may be very useful in locating abandoned private right of way, these maps often do not identify city streets (except for main thoroughfares). However, most cities over the years have had numerous maps published showing far more detail than is available on USGS topographic maps. One source of such maps is the David Rumsey Map Collection. This collection has thousands of maps. More importantly, they are georeferenced which means that any point on the map can be specified as a latitude and longitude coordinate. This in turn allows them to be displayed over Mapper's Google Maps or Open Street Map displays.

Since these maps are available on line, it is possible to display them as an overlay in Mapper. In order to do this, you must gather some information and setup the map as an overlay in Mapper.

Find a map

The first thing to do is to find a suitable map. The David Rumsey Map Collection link has a feature, the Luna Collection that allows you to browse through the collection of georeferenced maps. In the box in the upper right corner you can enter a place name like "Cincinnati, OHIO" Thumbnails of maps matching the search criteria will be displayed:


As an example, a 1901 map of Cincinnati has been located. Note that the thumbnail is highlighted.


View Map

Once you have located a suitable map, click on the thumbnail to select it then click on "VIEW IN GEOREFERENCER" to display the map as an overlay on an Open Streetmap base map. The selected map is displayed as an overlay on top of the Google Maps or Open Street Map. A slider control in the upper corner allows the opacity of the overlay to be varied. This is what Mapper can likewise do with this map overlay once it is configured.

Get Link

Now in the bar at the top, click on "This map". On the page that is displayed, there is a frame titled "Use in GIS apps" and click on "Get links". Note you must have previously registered and are logged on.


Paste Link data

In the box titled "Web Map Tile Service (WMTS)", copy the value to the clipboard. This is the HTML code that we need to extract some information to be entered into the Update Overlay Information dialog. To open the dialog, select Edit City Information and click on Add Overlay button. The paste the clipboard data into the lower box titled "Url:"


Edit Overlay Info

The dialog will now automatically fill in the other fields. If you wish, you may paste html into the Description field and you may have to edit the name to make it unique.


Press OK in the Update Overlay Information to save the overlay.

Save Overlay Info

Click on "OK" or "ADD" to add the new overlay to the list of overlays, The overlay can now be selected for a City in the Edit City Info dialog and then subsequently selected to be displayed from the Overlays.


Installing a map server on your PC.

Although these instructions are for installing a web server on your PC, they are generally applicable to any PC that will host a web sever. Here is a tutorial on how to Set Up Your Very Own Web Server!

Ubuntu (Linux)

Here are instructions for creating a web server on your Ubuntu PC: How to Install Apache, MySQL and PHP on Ubuntu 16.04. The step to install MySQL is not necessary for implementing a map server but the PHP is! However, installing MySQL does make it possible to store the Mapper database using a MySQL server.

The instructions call for installing a number of PHP add-ons. One add-on that you will definitely have to add is: which allows PHP to access SQLite3 databases.

Once you have installed the web server, you must create a directory "map_tiles" in the web server's root directory, "/var/www/html" into this directory, copy the file "mbtiles.php" from the ../mapper_app/Resources" directory. This is the code that will actually serve the map tiles.

Downloading the overlays

There are several ways to create overlay 'mbtiles' files including programs like MapTiler or mapping programs like QGIS or other mapping programs. However, I have already created a number of overlays for St Louis, Mo; Cincinnatti, OH; Louisville, KY and Berlin, Germany. They can be downloaded from Google Drive as zip files. They should be extracted into the "map_tiles" directory.
  1. St Louis
  2. St Louis 1912 topo
  3. Cincinnatti, Ohio
  4. Louisville, KY
A work of caution: These files are very large, so make sure you have plenty of disk space.

Once you have installed the mbtiles files in the "map_tiles"directory, enter this command in your web browser: "http://localhost/map_tiles/mbtiles.php"

This should result in a display like this:
